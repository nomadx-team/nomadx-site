Easy

Nomadx Elements are just vanilla web components, so styling and functionality is included with a single import. Getting started couldn't be easier.

Fast

Thanks to Stencil's performant renderer, Nomadx Elements are built for speed. Since there are no uneccesary updates to elements, performance should be lightning fast.

Accessible

There are a lot of UI Component Libraries out there, but few built with accessibility as their core focus. Every Nomadx Element is built to be accessible to assistive technology, clear and consistent, and up-to-standards.
